# Spa Framework

## To get started
1. Build the `node_modules` folder (if it does not exist)
	```shell
	npm install
	```
2. Start the Express server
	```shell
	node index.js
	```
	or, to start and open the page in your browser
	```shell
	npm test
	```
3. The page will be live at: <http://localhost:3000/>
